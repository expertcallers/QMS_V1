Email Monitoring form - Audit Guidelines


Customer Experience

Parameters Yes No
Associate used the standard greeting format Standard greeting format needs to be used. ( Client requirement) If the standard greeting format not used, this parameter will be rated down
Addressed the customer appropriately Addressed the customer correctly without shortening names (Eg., if the customer's name is 'Shivakumar', he should not be addressed as 'Shiv'). Also, honorifics like Mr., Mrs. to be used only with customer's full name or last name, never first name Parameter will be marked down if the name of the Customer is not addressed appropriately
Used appropriate grammar, spellings, punctuations and sentences with spacing Grammer and spell check to be applied before sending out emails. No short cuts/abbreviations to be used. Punctuations to be used when needed. Spacing between sentences and paragraphs to be maintained. If any of these parameters are not met, this will be marked down.
Probing is done as whenever it is needed Effective probing and questioning to extract relevant inforamtion from customers to be exhibited. Eg., alternate phone number, bank details, etc. Not probing and responding only with existing templates will be marked down
Assoicate outcalled the customer Out calling has to be done if further information/clarification is required, or if customer is not responding to emails If the outcalling is not done when it is required, this parameter would be rated down.
Associate used the standard closing format Has to use the standard closing with respective Phone/email Id as per the format Parameter will be marked down if standard closing format is not followed.


Business

Parameters Yes No
Accurate Resolution/Information is provided as per the process Associate to exhibit good understanding of the customer's query and process, resolution to be provided accordingly Not responding as per the query or providing misinformaton will be marked down
Specific format used to collect incomplete details Associate to use specific formats to collect incomplete details from customers. Not using appropriate formats will be marked down


Compliance

Parameters Yes No
Used standard and correct email template in all email communication Associate to use appropriate email templates based on subject matter Using wrong templates will be marked down
Resolution shared or moved ticket to the next level All communications must have relevant tickets that have been escalated to the right SPOCS or have been resolved Not creating tickets or not using the appropriate escalation metrics will be marked down
Disposed the ticket appropriately All tickets must be disposed based on resolution. Keeping tickets open will be marked down
